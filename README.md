
# Pokédex – Vanilla JS, HTML & CSS

Pokédex-Projekt mit Vanilla JavaScript, HTML und CSS – ohne Frameworks. Nutzt die öffentliche PokéAPI, unterstützt Suche/Filter, Paginierung.


## ⚡️ Features

🔎 Suche nach Namen

🏷️ Filter nach Typ (z. B. Fire, Water, Grass)

🔄 Paginierung (lädt weitere Pokémon)

🚀 Ohne Build-Tooling – funktioniert als reine Static Site


## 🧰 Tech-Stack
HTML5 - App Layout

CSS3 – CSS-Variablen, Flex/Grid

JavaScript (ES6+) – Fetch API, Module

PokéAPI – Datenquelle


## Installation

Install my-project with npm

```bash
  git clone https://github.com/realeif/pokedex.git
  cd pokedex
```
    
Danach per Live Server View starten.
